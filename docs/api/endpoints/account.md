# Account

Provides API for registering new users and other account management related functions.

**Operations:** 5 · [← back to index](../README.md)

| Method | Path | Summary |
| --- | --- | --- |
| POST | `/api/Account/ChangePassword` | Allows a logged in user to change the password. (Auth) |
| POST | `/api/Account/ConfirmEmail` |  |
| POST | `/api/Account/ForgotPassword` | Generates a password reset token and sends it to the email id of the user, if he is registered and his email id is confirmed. |
| POST | `/api/Account/ResetPassword` | Receives the password reset token generated by ForgotPassword API.  It resets the user's password with the newly specified password. |
| POST | `/api/Account/SendConfirmEmailToken` | Sends a token to the email id of the user.  The token is used to confirm his email id. |

## POST `/api/Account/ChangePassword`

Allows a logged in user to change the password. (Auth)

- **Operation ID:** `Account_ChangePassword_POST`
- **Auth:** Bearer token required

**Request body:** [`ChangePasswordReq`](../schemas.md#changepasswordreq)

**Responses**

| Status | Body |
| --- | --- |
| 200 — Success | [`MessageResponseDto`](../schemas.md#messageresponsedto) |
| 401 — Unauthorized | — |
| 403 — Forbidden | — |
